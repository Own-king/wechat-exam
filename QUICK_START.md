# 快速开始指南

## 1. 环境准备

### 1.1 安装微信开发者工具
- 访问 [微信开发者工具下载页](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
- 下载适合你操作系统的版本
- 安装并启动开发者工具

### 1.2 注册小程序账号
- 访问 [微信公众平台](https://mp.weixin.qq.com/)
- 注册小程序账号
- 获取 AppID（或使用测试号）

## 2. 导入项目

### 2.1 打开微信开发者工具
1. 启动微信开发者工具
2. 使用微信扫码登录

### 2.2 导入项目
1. 点击「导入项目」
2. 选择项目目录：`wechat-exam`
3. 填写项目信息：
   - 项目名称：实战云刷题小程序
   - 目录：选择 `wechat-exam` 文件夹
   - AppID：填入你的 AppID（或选择测试号）
4. 点击「导入」

## 3. 配置项目

### 3.1 修改 AppID
打开 `project.config.json`，修改 `appid` 字段：

```json
{
  "appid": "your-appid-here"
}
```

### 3.2 配置 API 地址
打开 `config/index.js`，修改 API 配置：

```javascript
const API_CONFIG = {
  [ENV.DEV]: {
    baseUrl: 'https://your-dev-api.com',  // 开发环境API地址
    timeout: 10000
  },
  [ENV.PROD]: {
    baseUrl: 'https://your-prod-api.com', // 生产环境API地址
    timeout: 10000
  }
}
```

### 3.3 准备图标资源
根据 `images/README.md` 中的说明，准备所需的图标资源：

#### 必需的图标（优先级高）：
1. **TabBar 图标**（8个）：
   - home.png / home-active.png
   - cert.png / cert-active.png
   - experiment.png / experiment-active.png
   - profile.png / profile-active.png

2. **基础图标**（3个）：
   - arrow-right.png（右箭头）
   - more.png（更多按钮）
   - scan.png（扫一扫）

#### 图标获取方式：
- 使用 [iconfont](https://www.iconfont.cn/) 下载图标
- 使用 [IconPark](https://iconpark.oceanengine.com/) 下载图标
- 使用设计工具自己制作

#### 图标规格：
- TabBar 图标：81px × 81px
- 功能图标：48px × 48px（@2x）
- 格式：PNG，支持透明背景

## 6. 常见问题

### Q1: 图标不显示怎么办？
**A:** 检查以下几点：
1. 图标文件是否已放入 `images/` 目录
2. 图标文件名是否正确
3. 图标路径是否正确（注意大小写）
4. 图标格式是否为 PNG

### Q2: 如何使用测试号？
**A:** 在导入项目时，AppID 选择「测试号」即可。测试号功能有限，但足够开发调试使用。

### Q3: 页面样式显示异常？
**A:** 检查：
1. 是否正确导入了所有文件
2. 微信开发者工具是否最新版本
3. 尝试重新编译项目

### Q4: 如何连接后端接口？
**A:** 
1. 修改 `config/index.js` 中的 API 地址
2. 在微信公众平台配置服务器域名白名单
3. 使用 `api/index.js` 中定义的接口方法

### Q5: 真机预览时网络请求失败？
**A:** 
1. 在微信公众平台配置合法域名
2. 开发阶段可以在开发者工具中勾选「不校验合法域名」
3. 确保手机网络正常
